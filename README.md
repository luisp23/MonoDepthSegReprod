# MonoDepthSegReprod
Final project for CS 7643 Deep Learning focusing on reproducing the results detailed in "Self-Supervised Monocular Scene Decomposition and Depth Estimation" - S. Safadoust,  F. Guney (2021)

## Dependencies (Current)
- `torchvision-0.11.1`
- `pytorch-1.10.0`
- `numpy-1.21.2`
